# proyectos_grupo_investigacion

Este repositorio contiene la estructura inicial de 3 proyectos distintos:

- `proyecto_calidad_agua`: proyecto basado en sensores para analizar distintos tipos de agua y mostrar sus cambios en un dashboard.
- `proyecto_emociones_vision`: proyecto de visión computacional para detectar rostros y clasificar emociones en tiempo real.
- `proyecto_smartstand_ble`: proyecto de analítica de interacción y aforo usando señales Bluetooth Low Energy (BLE).
